# GitHub Copilot in VS Code einrichten
### Schritt-für-Schritt-Anleitung für Einsteiger

---

## Schritt 1: Visual Studio Code herunterladen & installieren

1. Öffne deinen Browser und gehe auf **https://code.visualstudio.com**
2. Klicke auf den großen blauen **„Download"**-Button — Windows, Mac oder Linux wird automatisch erkannt
3. Die heruntergeladene Datei öffnen und der Installation folgen (einfach immer **„Weiter"** klicken, nichts verändern)
4. VS Code starten — ein dunkles Fenster öffnet sich

---

## Schritt 2: GitHub Copilot Extension installieren

1. Links in der Seitenleiste auf das **Extensions-Icon** klicken (sieht aus wie vier Quadrate, eines davon leicht versetzt)
2. Oben in die Suchleiste eintippen: **`GitHub Copilot`**
3. Beim ersten Suchergebnis (**„GitHub Copilot"** von GitHub) auf **„Install"** klicken
4. Kurz warten — die Extension wird automatisch installiert

> **Hinweis:** Es gibt auch „GitHub Copilot Chat" — das kann man ebenfalls installieren, ist aber nicht zwingend nötig für den Start.

---

## Schritt 3: Mit dem GitHub-Account anmelden

1. Unten links in der Statusleiste erscheint ein **Copilot-Icon** (kleines Sternchen/Funken-Symbol)
2. Darauf klicken → **„Bei GitHub anmelden"** wählen
3. Der Browser öffnet sich automatisch mit einer GitHub-Anmeldeseite
4. Mit dem **Firmen-GitHub-Account** (Sopra Steria) einloggen
5. Die Zugriffsberechtigung für VS Code **bestätigen**
6. Zurück in VS Code — Copilot ist jetzt aktiv

> Das Copilot-Icon unten links sollte jetzt **ohne rotes X** erscheinen.

---

## Copilot benutzen

**Copilot Chat (Fragen stellen):**
- Links in der Seitenleiste das **Chat-Icon** klicken (Sprechblase mit Sternchen)
- Oder Tastenkürzel: `Strg + Alt + I` (Windows) / `Ctrl + Cmd + I` (Mac)
- Einfach auf Deutsch fragen, z. B.: *„Erkläre mir diese Datei"* oder *„Schreib mir eine Funktion für..."*

**Automatische Vorschläge beim Tippen:**
- Eine Datei öffnen und anfangen zu schreiben
- Copilot macht grau hinterlegte Vorschläge
- **Tab** drücken zum Übernehmen, **Esc** zum Ablehnen

## Häufige Probleme

| Problem | Lösung |
|---|---|
| Copilot-Icon hat ein rotes X | Nicht angemeldet → Schritt 3 wiederholen |
| Keine Vorschläge beim Tippen | Unten rechts prüfen ob Copilot aktiv ist (kein X) |
| Anmeldung schlägt fehl | Prüfen ob der richtige GitHub-Account (Firmen-Account) verwendet wird |
| Extension nicht gefunden | Suchbegriff prüfen: genau `GitHub Copilot` eingeben |
